## Workflow в n8n
Название: **«Cardinal — Доставка»**
Статус: **Published** (опубликован)
Три узла:
1. **Webhook** — принимает POST запрос
2. **Code in JavaScript** — считает доставку по polyline маршрутов
3. **Respond to Webhook** — возвращает JSON

**Что принимает webhook (тело запроса):**
```json
{ "lat": 46.48, "lng": 30.73, "city": "Одеса, Одеська область, Україна" }
```

**Текущий код в n8n:** файл `n8n_delivery_v6.js`

**Polyline маршрутів:** 14 маршрутів, пересобрано з офіційних посилань заводу.
- Маршрут №12 включає Фастів (між Васильковом і Білою Церквою)
- Маршрут №14 — новий, Одеса (Жмеринка → Вінниця → Умань → Одеса → Миколаїв → Вознесенськ → Первомайськ → Умань → Жмеринка)
- Файл скрипта для оновлення polyline: `get_polylines_v3.py`

## Логика тарифов доставки
| Расстояние от маршрута | Статус | Цена |
|------------------------|--------|------|
| 0–2 км (город на маршруте) | on_route | 500–900 грн (диапазон) |
| до 15 км | deviation | 900 грн |
| 15–20 км | deviation | 1100 грн |
| 20–40 км | deviation | 1100 + 35×км |
| 40–45 км | clarify | Уточнення у менеджера |
| больше 45 км | nova_poshta | 4000 грн |
