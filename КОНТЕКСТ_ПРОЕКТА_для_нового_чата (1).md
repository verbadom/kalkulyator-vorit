## Workflow в n8n
Название: **«Cardinal — Доставка»**
Статус: **Published** (опубликован)
Три узла:
1. **Webhook** — принимает POST запрос
2. **Code in JavaScript** — считает доставку по polyline маршрутов
3. **Respond to Webhook** — возвращает JSON

**Адрес:** `http://localhost:5678/webhook/cardinal-delivery`

**Что принимает webhook (тело запроса):**
```json
{ "lat": 46.48, "lng": 30.73, "city": "Одеса, Одеська область, Україна" }
```
Геокодирование делается на фронтенде в браузере, в n8n передаются готовые координаты.

**Текущий код n8n:** версия 3.0 — принимает lat/lng, считает расстояние до polyline 13 маршрутов.

## Логика тарифов доставки
| Расстояние от маршрута | Статус | Цена |
|------------------------|--------|------|
| 0–2 км (город на маршруте) | on_route | 500–900 грн (диапазон) |
| до 15 км | deviation | 900 грн |
| 15–20 км | deviation | 1100 грн |
| 20–40 км | deviation | 1100 + 35×км |
| 40–45 км | clarify | Уточнення у менеджера |
| больше 45 км | nova_poshta | 4000 грн |

## Что нужно сделать — ГЛАВНАЯ ЗАДАЧА
- ⬜ **Google Places Autocomplete** в поле доставки

### Детали задачи Autocomplete:
Google с марта 2025 не даёт старый `google.maps.places.Autocomplete` новым аккаунтам.
Нужен новый `PlaceAutocompleteElement` ИЛИ альтернативное решение.

**Проблема которую решаем:** сейчас клиент вводит текст города вручную → фронтенд геокодирует через fetch к Google Geocoding API → получает координаты → передаёт в n8n. Это работает, но неудобно — нет подсказок, клиент может ошибиться в названии.

**Что хотим:** клиент начинает вводить название → появляются подсказки как в Google Maps → выбирает → координаты получаются точно.

**Ограничение:** только Украина (`componentRestrictions: { country: "ua" }`).

**Альтернативы которые стоит рассмотреть:**
1. `PlaceAutocompleteElement` (новый Google API) — пробовали, были проблемы с загрузкой
2. Nominatim (OpenStreetMap) — бесплатный, без ключа, украинские названия поддерживает
3. Algolia Places — платный
4. Самописный autocomplete через Geocoding API с debounce

**Рекомендация:** попробовать Nominatim — бесплатно, без ограничений по домену, хорошо знает украинские топонимы.
