# Проект: Калькулятор ворот завода «Кардинал» (Жмеринка)

## Кто и что

- **Завод:** Кардинал, г. Жмеринка
- **Алексей:** дилер завода в Одессе, сайт verbadom.com.ua
- **Платформа сайта:** Prom.ua (доступа к коду нет — калькулятор размещается на GitHub Pages)
- **Задача калькулятора:** Считать две вещи:
  1. Стоимость ворот (по модели, конфигурации, покрытию, размеру)
  2. Стоимость доставки по Украине от завода до клиента

---

## Технический стек

- **Frontend (сайт калькулятора):** GitHub Pages — простая страница с полями ввода
- **Backend (мозг):** n8n — установлен локально на компьютере Алексея
- **База данных:** Google Таблицы (две таблицы — цены и маршруты)
- **Карты:** Google Maps API — для определения расстояния до клиента
- **Архитектура:** «Черный ящик» — формулы скрыты в n8n, на сайте формул нет

## ⭐ Формулы расчёта цены ворот (найдены и зафиксированы)

### Принцип расчёта

Цены в Google Таблице — это цены **за погонный метр** ширины ворот.
Итоговая цена считается так:

```
Итоговая цена = CEILING(Цена завода × Коэффициент)
```

Функция CEILING — округление вверх до целого числа.

### Коэффициенты по комплектациям

| Комплектация | Коэффициент | Формула в Google Sheets |
|---|---|---|
| Ворота + Калитка отдельно | × 0.2325 | `=CEILING(B2*0,2325)` |
| Ворота с калиткой внутри (врезная) | × 0.2421 | `=CEILING(B2*0,2421)` |
| Только ворота (без калитки) | × 0.2427 | `=CEILING(B2*0,2427)` |

> Где B2 — ячейка с ценой завода в грн.

### Проверка (контрольный расчёт)

Модель «Стандарт», цена завода 16 500 грн:
- Ворота + калитка отдельно: 16 500 × 0.2325 = 3 836.25 → **3 837 грн** ✅
- Ворота с калиткой внутри: 16 500 × 0.2421 = 3 994.65 → **3 995 грн** ✅
- Только ворота: 16 500 × 0.2427 = 4 004.55 → **4 005 грн** ✅

Все три значения совпадают с таблицей «База Кардинал» — коэффициенты верные.

### Как эти коэффициенты используются в n8n

В узле **Code (JavaScript)** формула выглядит так:

```javascript
const factoryPrice = /* цена завода из Google Sheets */;

const priceGateWithSeparateWicket = Math.ceil(factoryPrice * 0.2325);
const priceGateWithBuiltInWicket  = Math.ceil(factoryPrice * 0.2421);
const priceGateOnly               = Math.ceil(factoryPrice * 0.2427);
```
