# Формули калькулятора воріт — Завод Кардинал (Жмеринка)
## Документ для внутрішнього використання. Складено на основі реверс-інжинірингу заводського калькулятора.
## Версія 2 — травень 2026. Всі формули перевірені на довільних прайсах.

---

## ЗАГАЛЬНІ ПРИНЦИПИ

### Округлення
- Всі формули використовують `ROUND()` — звичайне математичне округлення (не CEIL і не FLOOR)
- Фінальна ціна для клієнта: `CEIL(ціна / 100) × 100` — округлення вверх до сотні

### Стандартна ширина
- **Ковані ворота:** стандарт = **4.5м** (2 стулки по 1.8м + калітка 0.9м)
- **Сучасні ворота:** стандарт = **4.9м** (2 стулки по 2.0м + калітка 0.9м)

При стандартній ширині ціна = прайс (ціна дилера з таблиці).

### Ціни
Прайс — це ціна яку дилер виставляє в Google Таблиці. Клієнт ціну не вводить — вона береться з таблиці автоматично. Формули працюють для будь-якого прайсу.

## КОВАНІ ВОРОТА

### Варіант 1: Брама з калiткою (окремою)

```
if ширина < 4.5м:  ціна = прайс + 675
if ширина = 4.5м:  ціна = прайс
if ширина > 4.5м:  ціна = ROUND((прайс / 4.5 + 200) × ширина)
```

Приклади:
- прайс 13239, ширина 3.13м → 13239 + 675 = **13 914** ✅
- прайс 18100, ширина 4.5м → **18 100** ✅
- прайс 18100, ширина 5.0м → ROUND((18100/4.5 + 200) × 5) = **21 111** ✅
- прайс 24100, ширина 6.0м → ROUND((24100/4.5 + 200) × 6) = **33 333** ✅
- прайс 27654, ширина 5.0м → ROUND((27654/4.5 + 200) × 5) = **31 727** ✅

### Варіант 2: Брама без калітки

```
ціна = ROUND((прайс / 4.5 + 338) × ширина)
```

Повністю лінійна для будь-якої ширини. Немає стандарту, немає мінімалки.
Константа **338 = 200 + 138** (138 грн/м — надбавка за відсутність калітки).

Приклади:
- прайс 14729, ширина 3.18м → ROUND((14729/4.5 + 338) × 3.18) = **11 483** ✅
- прайс 18100, ширина 5.0м → ROUND((18100/4.5 + 338) × 5) = **21 801** ✅

### Варіант 3: Брама з калiткою всередині (врізна)

```
ціна = ROUND(прайс × 0.27778 × ширина)
```

Повністю лінійна. Немає стандарту, немає мінімалки.
Коефіцієнт **0.27778 = 5/18**

Приклади:
- прайс 18100, ширина 5.0м → ROUND(18100 × 0.27778 × 5) = **25 139** ✅
- прайс 21381, ширина 4.5м → ROUND(21381 × 0.27778 × 4.5) = **26 726** ✅
- прайс 21381, ширина 5.0м → ROUND(21381 × 0.27778 × 5) = **29 696** ✅
- прайс 22220, ширина 3.82м → ROUND(22220 × 0.27778 × 3.82) = **23 579** ✅

## СУЧАСНІ ВОРОТА

### Варіант 1: Брама з калiткою (окремою)

```
if ширина < 4.9м:  ціна = ROUND((прайс / 4.9 + 150) × 4.5)   ← мінімалка
if ширина = 4.9м:  ціна = прайс
if ширина > 4.9м:  ціна = ROUND((прайс / 4.9 + 200) × ширина)
```

**Мінімалка** рахується за формулою `ROUND((прайс / 4.9 + 150) × 4.5)` — універсальна для будь-якого прайсу і будь-якої зашивки. Окремо зберігати не потрібно.

Приклади:
- прайс 21800, ширина 3.0м → мін = ROUND((21800/4.9 + 150) × 4.5) = **20 695** ✅
- прайс 30500, ширина 3.0м → мін = ROUND((30500/4.9 + 150) × 4.5) = **28 685** ✅
- прайс 33000, ширина 3.0м → мін = ROUND((33000/4.9 + 150) × 4.5) = **30 981** ✅
- прайс 38900, ширина 3.0м → мін = ROUND((38900/4.9 + 150) × 4.5) = **36 399** ✅
- прайс 21800, ширина 4.9м → **21 800** (стандарт) ✅
- прайс 21800, ширина 5.0м → ROUND((21800/4.9 + 200) × 5) = **23 245** ✅
- прайс 34250, ширина 5.13м → ROUND((34250/4.9 + 200) × 5.13) = **36 884** ✅

### Варіант 2: Брама без калітки

```
ціна = ROUND((прайс / 4.9 + 338) × ширина)
```

Повністю лінійна. Немає стандарту, немає мінімалки.

Приклади:
- прайс 21800, ширина 5.0м → ROUND((21800/4.9 + 338) × 5) = **23 935** ✅
- прайс 21800, ширина 6.0м → ROUND((21800/4.9 + 338) × 6) = **28 722** ✅

### Варіант 3: Брама з калiткою всередині (врізна)

```
ціна = ROUND(прайс × 0.25 × ширина)
```

Повністю лінійна. Коефіцієнт **0.25 = 1/4**

Приклади:
- прайс 21800, ширина 5.0м → ROUND(21800 × 0.25 × 5) = **27 250** ✅
- прайс 25300, ширина 5.0м → ROUND(25300 × 0.25 × 5) = **31 625** ✅
- прайс 33450, ширина 5.0м → ROUND(33450 × 0.25 × 5) = **41 813** ✅

## ЗВЕДЕНА ТАБЛИЦЯ ФОРМУЛ

| Тип | Варіант | Стандарт | Мінімалка (при < стандарту) | Формула (при > стандарту) |
|-----|---------|----------|-----------------------------|---------------------------|
| Кована | З калiткою | 4.5м = прайс | прайс + 675 | ROUND((прайс/4.5 + 200) × ш) |
| Кована | Без калітки | немає | немає | ROUND((прайс/4.5 + 338) × ш) |
| Кована | З калiткою всередині | немає | немає | ROUND(прайс × 0.27778 × ш) |
| Сучасна | З калiткою | 4.9м = прайс | ROUND((прайс/4.9 + 150) × 4.5) | ROUND((прайс/4.9 + 200) × ш) |
| Сучасна | Без калітки | немає | немає | ROUND((прайс/4.9 + 338) × ш) |
| Сучасна | З калiткою всередині | немає | немає | ROUND(прайс × 0.25 × ш) |

де **ш** = ширина воріт у метрах.

## ЛОГІКА КОНСТАНТ

- **200** — базова надбавка за метр нестандартної ширини (однакова для кованих і сучасних)
- **138** — фіксована надбавка за відсутність калітки (однакова для кованих і сучасних)
- **338 = 200 + 138** — константа для варіанту "без калітки"
- **675** — мінімальна надбавка для кованих "з калiткою" при ширині < 4.5м
- **150** — константа в формулі мінімалки сучасних воріт
- **4.5** — множник в формулі мінімалки сучасних воріт (= стандартна ширина кованих)
- **0.27778 = 5/18** — коефіцієнт "всередині" для кованих
- **0.25 = 1/4** — коефіцієнт "всередині" для сучасних

## ФІНАЛЬНЕ ОКРУГЛЕННЯ ДЛЯ КЛІЄНТА

Після розрахунку ціни воріт — округлити вверх до сотні:

```javascript
finalPrice = Math.ceil(gatePrice / 100) * 100;
```

Приклад: 21 111 → **21 200**, 21 000 → **21 000**.

## ЩО НЕ ВХОДИТЬ В ЦЕЙ ДОКУМЕНТ

- Доставка (розраховується окремо через маршрути)
- Стовпи (фіксована ціна з таблиці)
- Фурнітура: замок з ручкою +1500 грн, засуви +600 грн
- Доплата за тип покриття: Матовий Гарантія +300 грн, Двосторонній Матовий +500 грн, Під дерево/3D +500 грн

## СТАТУС ПЕРЕВІРКИ

Всі формули перевірені порівнянням з заводським калькулятором на множині тестових точок, включаючи довільні прайси (не тільки стандартні з таблиці).
Точність: ±1 грн (float-артефакт JavaScript в заводському калькуляторі) — прийнятно.
Дата: травень 2026.
